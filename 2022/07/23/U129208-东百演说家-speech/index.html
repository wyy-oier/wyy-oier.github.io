阅读数 nan

$m,d\le1000$

直接暴力枚举每一天，看是否满足要求。

时间复杂度 $\mathcal{O}(md)$。

$w\le10^5$

设 $a’=a-1,b’=b-1$，则有同余方程

$dx+y\equiv a’\pmod{w}-①$

$dy+x\equiv b’\pmod{w}-②$

$x,y\in[0,\min(m,d)-1]$

那么我们枚举 $x%w$ 的取值，解出 $y%w$ 的取值，再用乘法原理计算方案数。

数据随机

用 $①\times d-②$ 得：$(d^2-1)x\equiv da-b\pmod{w}$

因为数据随机，不妨假设 $d^2-1\neq kw$，解出 $x\equiv x_0\pmod{w},y\equiv y_0\pmod{w}$，用乘法原理相乘即可。

无特殊情况

$(d+1)(d-1)\equiv0\pmod{w}$，分为两种情况。

  1. $d\equiv-1\pmod{w}$

    代入原方程，

    $-x+y\equiv a’\pmod{w}$

    $-y+x\equiv b’\pmod{w}$

    判一下 $a’+b’\equiv0\pmod{w}$，然后算 $-x+y\equiv a’\pmod{w}$ 的解。

    设 $cnt_i$ 表示 $[0,\min(m,d)-1]$ 内与 $i$ 同余的个数，用 set 维护数值相同的段。

  2. $d\equiv1\pmod{w}$

    代入原方程，

    $x+y\equiv a’\pmod{w}$

    $y+x\equiv b’\pmod{w}$

    判一下 $a’\equiv b’\pmod{w}$，然后算 $x+y\equiv a’\pmod{w}$ 的解。

    $x+y=x-(w-1-y)-1$，将问题转化为 $x-(w-1-y)\equiv a’+1\pmod{w}$，但要将 $x$ 对应的 set 的值翻转。
