Subtask 2+3

这是 $a_i \ne 0$ 的特殊情况，我们要证明一个结论：不将 $a$ 数组划分直接求得的一定是最优解。

证明:

设 $a$ 数组的 $n$ 个数分别为 $a_1,a_2,…a_n$，设最优解为 $ans$。

我们将 $a$ 数组划分为两段：$a_1 \sim a_x $，$a_{x+1} \sim a_n$。

假设 $a_1 \sim a_x$ 求得最优解 $ans_1$，此时 $i$ 的位置 $pos_1$，$a_{x+1} \sim a_n$ 求得最优解 $ans_2$，此时 $i$ 的位置 $pos_2$。

那么我们就转化为了：证明 $ans \le ans_1+ans_2$。

由题意可知：

  • $ans_1$ 可向周围最多延伸 $(ans_1-1)$ 格
  • $ans_2$ 可向周围最多延伸 $(ans_2-1)$ 格

那么 $pos_1$ 与 $pos_2$ 之间距离最多为 $dis=ans_1+ans_2-2$。

这一段对答案的贡献为 $\lfloor \dfrac{ans_1+ans_2-2}{2} \rfloor+1=\lfloor\dfrac{ans_1+ans_2}{2} \rfloor$。

则 $ans=\min(\max(ans_1,ans_2+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor),\max(ans_1+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor,ans_2))$

我们分类讨论 $ans$ 取值情况。

① $ans=ans_1$ 或 $ans_2$，显然 $ans<ans_1+an_2$。

② $ans=ans_1+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor$

有不等式 $ans_1+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor \le ans_2+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor $

得 $ans_1\le ans_2$

则 $ans=ans_1+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor \le ans_1+\dfrac{2ans_2}{2}=ans_1+ans_2$

③ $ans=ans_2+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor$

同理，得 $ans_1 \ge ans_2$

$ans=ans_2+\lfloor\dfrac{ans_1+ans_2}{2} \rfloor \le ans_2+\dfrac{2ans_1}{2}=ans_1+ans_2$

综上，当$a_i \ne 0$，$ans\le ans_1+ans_2$，不分段求解最优。

那么接下来这两个 Subtask 的问题就转化为了：求 $\sum\limits_{i=1}^{n}\min(\sum\limits_{j=1}^{n}\max(a_j+\text{abs}(i-j)))$。

$O(n^2)$ Code

下面是 $O(n)$ 的写法：

观察要求式子 $a_j+\text{abs}(i-j)$，可将其分为两种：$a_j+j-i$ 或 $a_j-j+i$

于是想到预处理 $a_i+i$ 与 $a_i-i$

从后往前得出 $\max(a_i+i)$ ，从前往后得出 $\max(a_i-i)$。

答案为 $\sum\limits_{i=1}^n\min{\max(dis1_i-i,dis2_{i-1}+i)}$。

Subtask4

此时 $a_i=0$，但 $n\le 10^3$，考虑区间DP。

首先，以$0$为间隔，将整个 $a$ 数组分为几段使得每一段都是由连续正整数构成的。

然后定义 $dp[l][r]$ 为 $l$ 段至 $r$ 段的最优解。

初始化为 $dp[i][i]$ 为第 $i$ 小段的最优解，由于都是正整数，直接 $\text{cal}()$ 即可。

转移方程为 $dp[l][r]=\min(\text{cal(),dp[l][i]+dp[i+1][r]})$。
